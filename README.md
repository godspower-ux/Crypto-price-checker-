# Crypto Price Checker 🪙

A Python program that fetches live cryptocurrency prices using the CoinGecko API.

## Features
- Get real-time price of any cryptocurrency
- Supports multiple currencies (USD, EUR, GBP)
- Error handling for invalid coin names
- Loop that lets you check multiple coins without restarting

## Tools Used
- Python
- Requests library
- CoinGecko API (free, no API key needed)

## How It Works
The user enters a coin name and currency, the program fetches
live data from CoinGecko's API, parses the JSON response,
and displays the current price cleanly.
